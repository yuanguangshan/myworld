## 游戏概述与设计理念

### 什么是 MyWorld？

**MyWorld**（现更名为 **RPG World - Action RPG Maker**）是一款革命性的 3D 动作 RPG 创作工具。它独特地融合了两个截然不同的体验：

- **创作者模式**：无代码、可视化地构建 3D 动作 RPG 世界
- **玩家模式**：即时游玩自己创作的内容或社区玩家的作品

### 核心设计概念

MyWorld 的核心理念是 **"模糊创造与游玩的界限"**：

> "在同一个 3D 环境中，你既是冒险者，也是创造者。"

传统游戏开发工具需要编程、学习复杂引擎，而 MyWorld 提供了：
- **拖放式编辑器**：350+ 可用道具，一键放置
- **实时地形雕刻**：无需建模工具即可塑造地形
- **模块化敌人生成**：混合搭配身体部件创建独特敌人
- **无缝多人游戏**：原生支持最多 4 人合作或对战

### 目标受众

MyWorld 专为以下群体设计：

| 玩家类型 | 需求 | MyWorld 如何满足 |
|----------|------|-----------------|
| **创意爱好者** (15-35岁) | 想要创造但缺乏编程技能 | 无代码编辑器，即时视觉反馈 |
| **动作 RPG 玩家** (18-40岁) | 喜欢暗黑破坏神、黑暗之魂类游戏 | 熟悉的战斗节奏，合作游戏 |
| **教育工作者** (25-50岁) | 教授游戏设计 | 可视化工具，项目式学习 |

### 市场定位

MyWorld 在创作者工具生态中占据独特位置：

- **vs RPG Maker**：3D 实时战斗 vs 2D 回合制战斗，更易上手但深度略逊
- **vs Dreams**：专注于动作 RPG，学习曲线更低，但通用性不如 Dreams
- **vs Roblox**：无需 Lua 脚本即可创作，但社区规模较小
- **vs Unity/Unreal**：专业引擎 vs 爱好者工具，MyWorld 降低门槛但牺牲灵活性

## 游戏系统详解

### 🎮 战斗系统

MyWorld 采用经典动作 RPG 战斗机制，强调时机掌握和资源管理。

#### 基础战斗机制

| 动作 | 操作 | 说明 |
|------|------|------|
| **近战攻击** | 鼠标左键 | 3 段连击，持续点击触发 |
| **远程攻击** | 鼠标右键 | 长距离横扫 |
| **格挡** | `\` (反斜杠) | 消耗体力，仅前方有效 |
| **翻滚闪避** | `Ctrl` 或双击方向 | 无敌帧，躲避攻击 |
| **锁定目标** | `Tab` | 自动追踪最近敌人 |
| **冲刺** | `Shift` | 提高移动速度 |
| **技能 1-4** | `1` `2` `3` `4` | 四个特殊能力，有冷却时间 |
| **喝药水** | `Alt` | 恢复生命值 |

#### 战斗策略

**新手技巧**：
1. **善用锁定系统** - 使用 `Tab` 锁定敌人，避免方向迷失
2. **管理体力** - 格挡消耗体力，不要持续按住
3. **连击节奏** - 3 段连击有时间窗口，过快或过慢都会中断
4. **利用翻滚** - Boss 攻击前摇明显，及时翻滚躲避

**进阶技巧**：
1. **破坏环境** - 大部分物体可破坏，可能隐藏物品
2. **技能组合** - 研究不同技能的连招，最大化伤害
3. **风筝战术** - 利用远程攻击配合翻滚，远程风筝敌人
4. **团队配合** - 多人游戏中分配角色（坦克、输出、辅助）

#### 战斗反馈

- **视觉反馈**：命中屏幕震动、伤害数字、敌人受击动画
- **音效反馈**：命中音效、暴击音效、技能音效
- **状态效果**：中毒（持久）、减速、击退等状态

### 📋 任务系统

MyWorld 的任务系统简单直接，专注于动作 RPG 的核心循环。

#### 任务类型

| 类型 | 描述 | 示例 |
|------|------|------|
| **击杀任务** | 消灭指定数量敌人 | "击败 10 只哥布林" |
| **收集任务** | 收集指定物品 | "采集 5 个蘑菇" |
| **任务链** | 多个任务相互关联 | 先收集材料，再击败 Boss |

#### 任务机制

**任务接受**：
- 与 NPC 对话（按 `U` 键交互）
- 查看任务追踪器（按 `L` 键）
- 任务显示目标、进度、奖励

**任务完成**：
- 返回 NPC 交付
- 获得经验值和金币
- 解锁后续任务或传送门

**任务难度与奖励**：
- **难度越高，XP 越多** - 困难任务提供更多经验
- **XP 用于技能分配** - 提升角色属性
- **金币购买装备** - 从 NPC 处购买武器和防具

#### 传送门解锁条件

传送门是区域间的主要通行方式，常见解锁条件：

- **击杀所有敌人** - 清理当前世界所有怪物
- **完成所有任务** - 完成当前世界所有 NPC 任务
- **组合条件** - 既要求击杀也要求完成任务
- **特定任务覆盖** - 单个任务解锁传送门，无视其他条件

### 📈 角色成长系统

MyWorld 提供多种成长路径，满足不同玩家偏好。

#### 属性系统

角色可提升的属性包括：

| 属性 | 作用 | 升级效果 |
|------|------|----------|
| **生命值** | 承受伤害的能力 | 提高最大生命值 |
| **体力** | 格挡和翻滚消耗 | 提高体力上限和恢复速度 |
| **幸运** | 掉落率和暴击率 | 提高稀有物品获取 |
| **力量** | 近战伤害 | 提高武器伤害 |
| **敏捷** | 攻击速度和移动 | 提高攻击频率和移动速度 |

#### 技能图表

- **XP 获取**：完成任务、击败敌人获得
- **技能分配**：自由分配 XP 到不同属性
- **重置技能**（如支持）：重新分配技能点

#### 装备系统

- **武器**：近战武器（剑、斧）、远程武器（弓）
- **防具**：头盔、胸甲、护腿、护靴
- **饰品**：戒指、项链提供特殊加成
- **传说装备**：Creative Pack 增加传说物品

#### 职业选择

MyWorld 预设多个职业，每个职业有独特技能：

| 职业 | 特点 | 适用玩家 |
|------|------|----------|
| **战士** | 高生命，近战专注 | 喜欢正面硬刚 |
| **法师** | 远程法术，AOE 伤害 | 站桩输出爱好者 |
| **盗贼** | 高机动，暴击伤害 | 灵活操作型 |
| **猎人** | 远程物理，宠物辅助 | 生存优先玩家 |

### 👥 多人游戏系统

MyWorld 内置多人游戏支持，无需复杂设置。

#### 合作模式 (Co-op)

- **最大 4 人** - 最多 4 名玩家合作探索
- **共享进度** - 任务完成、传送门解锁对所有人生效
- **难度调整** - 玩家越多，敌人越强
- **战利品分配** - 每人独立掉落，无抢夺

#### 玩家对战 (PvP)

MyWorld 支持两种 PvP 模式：

| 模式 | 描述 | 刷新点 |
|------|------|--------|
| **大逃杀** | 所有玩家在同一个刷新点 | 单一生成点 |
| **红蓝对抗** | 分队死亡竞技 | 红队/蓝队各自刷新点 |

**PvP 操作**：
- `F2` - 开始 PvP 比赛（仅主机）
- `Enter` - 打开/关闭聊天

#### Steam Workshop 集成

MyWorld 从设计之初就围绕 Steam Workshop 构建：

- **上传世界** - 编辑模式按 `F5` 上传
- **下载世界** - 通过 Browse 门户订阅
- **自动打包** - 关联的多个世界自动打包上传
- **图片预览** - 必须小于 500KB，最大 512x512px

## 世界创建指南

MyWorld 的核心魅力在于其直观的世界创建工具。

### 🏰 Hubworld 与传送门

#### Hubworld 空间菜单

MyWorld 创新性地使用 3D Hubworld 作为主菜单：

**Hubworld 功能**：
- **创造门户** - 进入世界编辑器
- **游玩门户** - 选择并游玩世界
- **浏览门户** - 访问 Steam Workshop
- **主机门户** - 主持多人游戏
- **RPG World TV** - 观看内置教程视频
- **角色自定义** - 更换英雄角色

**传送门导航**：
- 传送门以圆形排列在 Hubworld 中
- 进入传送门 = 选择功能
- 物理探索代替 2D 菜单导航
- 更具沉浸感和探索乐趣

#### 传送门链接系统

MyWorld 最强大的创新是**传送门链接**，将多个小世界连接成大型 RPG 冒险。

**链接方式**：
1. 在世界中放置传送门
2. 配置传送门目标（链接到其他世界）
3. 设置解锁条件（击杀/完成/组合）
4. 上传时自动打包所有关联世界

**创作模式**：
- **线性冒险** - World A → World B → World C
- **辐射设计** - 中心世界连接多个子世界
- **网状结构** - 多个世界相互链接

### 🗺️ 地形雕刻工具

MyWorld 提供六种地形雕刻模式，实时、直观地塑造地形。

#### 编辑模式按键

| 模式 | 按键 | 功能 |
|------|------|------|
| **对象模式** | `1` | 放置和操作对象 |
| **地形绘制** | `2` | 绘制地形纹理 |
| **提升/降低** | `3` | 雕刻地形（升高或降低） |
| **平整地形** | `4` | 将地形平整到选定高度 |
| **平滑地形** | `5` | 平滑粗糙地形边缘 |
| **创建坡道** | `6` | 创建连接高度的坡道 |

**编辑模式通用操作**：
- `[` `]` - 调整笔刷大小
- `Z` - 撤销上一步操作
- `Delete` - 删除选定对象
- `R` - 旋转对象
- `Shift` + `R` - 旋转 90 度

#### 地形雕刻技巧

**新手建议**：
1. **从小开始** - 先创建小型世界熟悉工具
2. **使用预览模式** - 频繁切换游玩模式测试
3. **分层创建** - 先地形，后建筑，再敌人
4. **善用平滑** - 雕刻后用模式 5 平滑边缘

**进阶技巧**：
1. **创建河床** - 用模式 4 平整底部，模式 3 雕刻河岸
2. **建造山脉** - 模式 3 逐渐升高，模式 5 平滑山坡
3. **地下洞穴** - 模式 4 平整天花板，模式 3 雕刻地下结构
4. **多高度平台** - 模式 6 创建坡道连接不同高度

### 🎨 对象放置与世界绘制器

#### 对象放置 (模式 1)

MyWorld 提供 350+ 内置对象：

| 类别 | 示例 |
|------|------|
| **建筑** - 酒馆、房屋、城墙、塔楼 |
| **装饰** - 桶、花瓶、树木、石头 |
| **NPC** - 商人、任务发布者、友方角色 |
| **敌人** - 骷髅、诅咒骑士、Boss |
| **特殊** - 鸡（是的，鸡！）、家具、灯光 |

**放置操作**：
- 从对象库拖放对象到世界
- 选择对象后可移动、旋转、缩放
- `R` 旋转，`Z` 撤销，`Delete` 删除

#### 世界绘制器 (World Painter)

世界绘制器是 MyWorld 的强大批量放置工具：

- **一笔绘制** - 用笔刷快速覆盖大面积区域
- **智能密度** - 自动调整对象密度，避免重叠
- **主题一致** - 保持视觉风格统一
- **效率提升** - 比逐个放置快 10-100 倍

**使用场景**：
- 快速创建森林（拖放树木）
- 铺设城镇街道（放置路灯、装饰）
- 添加环境细节（石头、植被）

### 👾 敌人创建器

MyWorld 的敌人创建器让你无需 3D 建模技能即可创建独特敌人。

#### 模块化敌人系统

**身体部件混合**：
- **头部** - 南瓜头、巨人眼球、骷髅头等
- **躯干** - 机器人身体、怪物躯干等
- **左臂** - 机械臂、骨手等
- **右臂** - 可与左臂不同，创造不对称敌人

**视觉自定义**：
- **调色板** - 自定义身体部位颜色
- **发光效果** - 添加眼睛、魔法发光
- **金属质感** - 赋予盔甲金属光泽

**行为与数值**：
- **生命值** - 设置敌人耐久度
- **伤害** - 攻击力数值
- **体型** - 缩放敌人大小
- **行为模式** - 选择 AI 行为（巡逻、追击、守护）

#### 高级敌人创建技巧

**创建漂浮敌人**：
1. 隐藏腿部身体部件
2. 调整漂浮高度
3. 添加发光效果（如幽灵）

**Boss 设计**：
1. 巨大体型（2-3 倍）
2. 高生命值和伤害
3. 特殊行为模式
4. 专属掉落奖励

**敌人多样性**：
- 用不同部件创造不同敌人类型
- 同一区域避免重复敌人
- 使用主题一致的颜色方案

### 🔗 世界链接最佳实践

#### 链接策略

**线性章节式**：
```
世界 1（新手村）
  ↓ 传送门（完成任务）
世界 2（冒险区域）
  ↓ 传送门（击败 Boss）
世界 3（魔王城）
  ↓ 传送门（通关）
```

**辐射中心式**：
```
      世界 A      世界 B
          \      /
            Hubworld
          /      \
      世界 C      世界 D
```

**网状探索式**：
```
世界 A ↔ 世界 B
  ↕        ↕
世界 C ↔ 世界 D
```

#### 解锁条件设计

**渐进解锁**：
- 早期世界：简单击杀任务
- 中期世界：任务链 + 击杀 Boss
- 后期世界：组合条件 + 高难度

**可选区域**：
- 隐藏区域：高难度，但非主线必需
- 奖励区域：提供稀有装备或特殊技能

#### 上传 Workshop

**准备工作**：
1. 确保所有关联世界已链接正确
2. 在第一世界按 `F5` 上传
3. 系统自动打包所有关联世界
4. 添加预览图片（小于 500KB）

**命名规范**：
- 清晰的标题（如"暗黑森林冒险 第一章"）
- 描述包含章节信息、难度、游玩时间
- 标签用于分类（如"单人"、"多人"、"困难"）

## 创新特性解析

MyWorld 在 RPG 创作者工具领域引入了多项真正创新的功能。

### 1️⃣ Hubworld 空间菜单

**创新点**：将传统 2D 菜单转换为沉浸式 3D 空间

**优势**：
- **直观导航** - 物理进入传送门而非点击菜单
- **发现感** - 看到其他玩家的世界创造物，激发探索欲
- **记忆更容易** - "黑暗城堡世界通过阴森传送门"比"世界 ID: 48291"好记
- **沉浸感** - 创造和游玩在同一 3D 环境中

**为什么重要**：降低了 UGC（用户生成内容）的浏览摩擦，对社区网络效应至关重要。

### 2️⃣ 传送门链接系统

**创新点**：用户可以将多个小世界链接成连贯的 RPG 冒险

**独特价值**：
- **章节式冒险** - 创造有开头、中段、结尾的完整故事
- **协作创作** - 多人各自创建世界，然后链接成大型冒险
- **社区宇宙** - 玩家们将各自世界连接成共享宇宙

**对比**：
- **RPG Maker** - 需要脚本知识，学习曲线陡峭
- **Roblox** - 游戏间链接需要编程
- **MyWorld** - 原生支持，一键链接，无代码

### 3️⃣ 模块化敌人生成器

**创新点**：在游戏引擎内通过混合身体部件创建敌人，无需外部 3D 建模工具

**革命性改变**：
- **传统流程**：Maya/Blender 建模 → 导出 → 导入引擎 → 调整（数小时/天）
- **MyWorld 流程**：选头 + 选身体 + 选四肢 → 调色 → 完成（数分钟）

**创作自由**：
- 无限组合（理论上）
- 支持漂浮、不对称、巨大体型等特殊设计
- 即时预览，快速迭代

### 4️⃣ 原生 UGC 多人游戏

**创新点**：从零开始就为**用户创建的内容**内置多人游戏支持

**区别**：
- **大多数创作者工具** - 仅官方内容支持多人，UGC 需要复杂设置或完全不支持
- **MyWorld** - 每个用户世界原生支持最多 4 人合作/PvP

**影响**：
- 降低社交门槛 - 朋友间直接邀请游玩
- 提高内容重玩性 - 合作探索比单人更有趣
- 增强传播性 - "我和朋友玩了这个世界，你也试试"

### 5️⃣ 实时地形雕刻

**创新点**：在 3D 空间中实时雕刻地形，无需顶点编辑器

**技术优势**：
- **即时反馈** - 雕刻操作立即看到效果
- **直观操作** - 笔刷式操作，所见即所得
- **高效流程** - 无需导入/导出地形网格

**学习曲线**：
- 相比传统顶点编辑器，MyWorld 的 6 模式系统对非专业人士友好
- 新手 10 分钟内可创建有起伏的地形
- 传统工具可能需要数小时学习

### 6️⃣ Steam Workshop 优先架构

**创新点**：从设计第一天就将 Steam Workshop 作为核心功能，而非后加的附加

**体现**：
- **无缝上传** - 编辑器内置 `F5` 上传功能
- **自动打包** - 链接的多个世界自动打包
- **内置浏览** - Hubworld 的 Browse 门户直接访问 Workshop
- **订阅系统** - 一键订阅，自动下载更新

**为什么重要**：
- UGC 生态依赖分享的便捷性
- MyWorld 将分享流程降到最低摩擦
- 创作者上传 → 玩家下载 → 玩家成为创作者的飞轮

## 游戏技巧与策略

### 🎯 创作者技巧

#### 世界设计原则

**1. 从小开始，逐步扩展**

❌ **错误**：
- 第一个世界就创建 50 个传送门
- 试图建造整个王国

✅ **正确**：
- 先创建 1 个小型村庄（3-5 栋建筑）
- 添加 1-2 个 NPC 和基础任务
- 测试游玩，收集反馈

**2. 流程优先，装饰其次**

❌ **错误**：
- 花费 80% 时间摆放家具、花草
- 最后才考虑任务和敌人

✅ **正确**：
- 60% 时间设计流程、任务、敌人位置
- 20% 时间地形和建筑
- 20% 时间装饰和细节

**3. 垂直利用空间**

MyWorld 没有跳跃机制，但可以通过其他方式利用垂直空间：

- **使用坡道**（模式 6）连接不同高度
- **创建多层建筑** - 通过楼梯和坡道
- **高地优势** - 远程攻击的制高点
- **地下区域** - 雕刻地下洞穴和密室

#### 地形雕刻技巧

**创建河流**：
1. 用模式 4 平整河床到统一高度
2. 用模式 3（降低）雕刻出河道
3. 用模式 5 平滑河岸
4. 用模式 2 添加水面纹理

**建造山脉**：
1. 用模式 3 逐渐升高中心区域
2. 用模式 5 平滑山坡，避免尖角
3. 用模式 6 创建登山路径
4. 在山巅放置奖励或传送门

**创建峡谷**：
1. 用模式 3 降低两个山峰之间的区域
2. 用模式 5 平滑峡谷壁
3. 添加桥或绳索连接两岸

#### 敌人布局

**难度曲线**：
- **新手区** - 1-2 个弱敌人，间隔远
- **中期区** - 3-4 个中等敌人，小群组
- **后期区** - Boss + 小怪群，密集战斗

**战斗多样性**：
- **开阔地** - 远程敌人 + 近战护卫
- **狭窄通道** - 近战伏击
- **高台** - 远程弓箭手压制

**奖励放置**：
- 敌人附近放置宝箱（高风险高回报）
- 隐藏区域放置稀有物品
- Boss 必掉落传说装备

### ⚔️ 玩家技巧

#### 战斗策略

**新手生存指南**：

1. **善用锁定系统**
   - 按 `Tab` 锁定敌人
   - 避免迷失方向
   - 专注目标节奏

2. **管理体力**
   - 格挡消耗体力，不要一直按住
   - 翻滚比格挡更省体力
   - 体力耗尽时无法攻击或防御

3. **3 段连击节奏**
   - 点击 → 暂停 → 点击 → 暂停 → 点击
   - 点击过快会打断连击
   - 点击过慢会重置连击

4. **优先远程敌人**
   - 弓箭手和法师优先消灭
   - 避免被远程风筝
   - 使用远程攻击或翻滚接近

**Boss 战技巧**：

1. **观察攻击模式**
   - Boss 攻击前摇明显（抬手、吼叫）
   - 熟悉攻击频率和范围
   - 预判攻击，提前翻滚

2. **利用破坏环境**
   - 大部分 Boss 所在地有可破坏物体
   - 摧毁可能掉落生命药水
   - 某些环境可阻挡 Boss 移动

3. **团队配合**
   - **坦克** - 吸引 Boss 注意力
   - **输出** - Boss 后方或侧翼输出
   - **辅助** - 投掷药水或干扰 Boss

4. **不要贪刀**
   - 打两三下就翻滚远离
   - 不要贪图最后一击被 Boss 秒杀
   - 稳健比冒险更重要

#### 任务完成技巧

**效率最大化**：

1. **任务批量处理**
   - 一次性接受多个任务
   - 规划路线，顺路完成
   - 减少往返 NPC 次数

2. **优先高价值任务**
   - 高难度任务 = 更多 XP
   - 提前查看任务奖励
   - 优先高 XP 任务

3. **传送门条件管理**
   - 击杀任务 + 收集任务同时完成
   - 不要漏掉任何敌人
   - 仔细搜索每个区域

**任务链技巧**：

1. **理解依赖关系**
   - 有些任务需要先完成前置任务
   - 观察任务描述中的提示
   - 完成前置任务再继续

2. **利用存档**
   - 在关键选择前存档
   - 尝试不同分支（如果有）
   - 找到最佳奖励路径

#### 探索与收集

**探索技巧**：

1. **破坏所有物体**
   - 桶、花瓶、石头柱都可破坏
   - 隐藏药水或装备
   - 某些物体阻挡隐藏道路

2. **检查每个角落**
   - 墙后、桥下、洞穴内
   - 高地和低地都要探索
   - 隐藏区域常有稀有物品

3. **记住回程路**
   - 深入探索前记住路径
   - 标记关键地点（如传送门、Boss 位置）
   - 使用传送门快速回到 Hubworld

**收集策略**：

1. **优先稀有资源**
   - 某些材料在特定区域才有
   - 不要随意消耗稀有物品
   - 保留给关键任务

2. **交易策略**
   - 与多个 NPC 交易比价
   - 某些 NPC 专精特定物品
   - 低买高卖赚取金币

### 🚫 常见错误避免

#### 创作者错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| **第一个世界太大** | 难以完成，质量低劣 | 从小世界开始（1-2 栋建筑） |
| **只注重装饰** | 玩家很快无聊 | 先设计玩法和任务，再装饰 |
| **难度不平衡** | 玩家放弃 | 测试游玩，调整敌人数量和强度 |
| **任务类型单一** | 重复感强 | 混合击杀、收集、探索任务 |
| **没有指引** | 玩家迷路 | 使用地标、路牌、NPC 提示 |
| **免费版上传受限** | 无法分享社区 | 购买 Creative Pack 解锁 |

#### 玩家错误

| 错误 | 后果 | 正确做法 |
|------|------|----------|
| **不使用锁定** | 攻击落空 | 按 `Tab` 锁定敌人 |
| **体力耗尽时战斗** | 无法格挡，容易被杀 | 留体力用于关键时刻 |
| **贪刀不舍得撤退** | 被 Boss 秒杀 | 打几刀就翻滚远离 |
| **忽略任务追踪** | 遗漏目标 | 按 `L` 键查看任务进度 |
| **不探索隐藏区域** | 错过稀有物品 | 破坏所有物体，检查角落 |
| **单人玩合作任务** - 难度过高 | 邀请朋友合作或降低难度 |

### 🎓 进阶技巧

#### 创作者高级技术

**事件逻辑创意**：
- 虽然没有完整脚本系统，但可通过传送门条件模拟
- **Boss 通关解锁** - Boss 所在地传送门条件设置为"击败所有敌人"
- **剧情分支** - 多个 NPC 给出不同任务，选择不同路径
- **隐藏触发** - 某个物体破坏后，新传送门出现

**世界节奏控制**：
- **开阔战斗区** - 给玩家空间躲避
- **狭窄通道** - 增加紧张感
- **高处眺望** - 展示接下来区域，激发探索欲

**主题一致性**：
- 使用同主题的敌人颜色
- 建筑风格与环境匹配
- NPC 外观符合世界观

#### 玩家高阶技术

**连招优化**：
- 研究武器连招窗口
- 技能冷却时使用普通攻击
- 技能穿插普通攻击最大化 DPS

**地形利用**：
- 高地打远程，减少被击中
- 窄道限制敌人移动，逐个击破
- 环境物体阻挡敌人进攻

**经济优化**：
- 不买不必要的装备
- 优先升级武器而非防具
- 等高等级再买昂贵装备

## 常见问题解答

### ❓ 技术问题

**Q: MyWorld 支持手柄吗？**

A: 不支持。MyWorld 目前仅支持键盘和鼠标操作，这是游戏的主要限制之一。未来可能会添加手柄支持。

**Q: 游戏支持哪些操作系统？**

A: 仅支持 Windows (PC)。暂无 macOS 或 Linux 版本，这是由游戏引擎决定的。

**Q: 游戏会跳跃吗？**

A: 不会。MyWorld 没有跳跃机制，这可能导致一些玩家感到受限。地形设计时需要使用坡道连接不同高度。

**Q: 游戏有中文支持吗？**

A: 取决于版本。请检查 Steam 商店页面是否列出了中文语言支持。如果未列出，可能需要等待官方更新或使用社区翻译。

**Q: 游戏卡顿怎么办？**

A: 尝试以下方法：
1. 降低游戏画质设置（按 `F11` 进入设置）
2. 减少世界中的对象数量
3. 关闭后台程序
4. 更新显卡驱动

### 📦 Workshop 上传问题

**Q: 免费版可以上传到 Workshop 吗？**

A: 不可以。免费版限制：
- 最多创建 2 个世界
- 无法链接多个世界
- 无法上传到 Steam Workshop

需要购买 **Creative Pack DLC** 解锁这些功能。

**Q: 上传时提示"图片太大"怎么办？**

A: Workshop 预览图片必须：
- 小于 500KB
- 最大 512x512 像素
- 推荐使用 PNG 或 JPG 格式

**Q: 链接的世界怎么上传？**

A: 系统会自动处理：
1. 在链接世界的第一个世界中按 `F5`
2. 系统自动打包所有关联世界
3. 一次性上传整个世界链

**Q: 上传后能修改吗？**

A: 可以。更新世界：
1. 在编辑模式下修改世界
2. 再次按 `F5` 上传
3. 选择覆盖现有世界或创建新版本

### 🎮 游戏玩法问题

**Q: 任务无法完成怎么办？**

A: 尝试以下方法：
1. 按 `L` 查看任务追踪器，确认目标
2. 仔细检查任务描述中的要求
3. 尝试重新进入世界（退出再进入）
4. 如果是 Bug，报告给开发者或社区

**Q: Boss 太强打不过怎么办？**

A: 几个建议：
1. 升级角色属性（完成其他任务获取 XP）
2. 购买更好的装备
3. 邀请朋友合作（最多 4 人）
4. 研究Boss 攻击模式，躲避攻击
5. 利用地形优势（高处、窄道）

**Q: 如何自定义敌人？**

A: 需要 **Creative Pack DLC**：
1. 进入编辑模式
2. 选择敌人创建器
3. 混合身体部件、调整属性
4. 保存并放置到世界中

**Q: 可以添加自定义 3D 模型吗？**

A: 目前不支持。MyWorld 不允许导入外部 3D 模型（如 Blender 创建的模型）。这是社区最常请求的功能之一，但截至 2026 年仍未实现。

**Q: 任务类型有限怎么办？**

A: 确实，MyWorld 目前只支持击杀和收集任务。这是游戏的主要限制之一。开发者表示未来可能添加更多任务类型。

### 🏢 版本与 DLC 问题

**Q: 免费版和付费版有什么区别？**

A: 主要区别：

| 功能 | 免费版 | Creative Pack |
|------|--------|---------------|
| 创建世界数量 | 最多 2 个 | 无限制 |
| Workshop 上传 | ❌ | ✅ |
| 世界链接 | ❌ | ✅ |
| 敌人创建器 | ❌ | ✅ |
| 对象库 | 350+ | 900+ |
| 世界主题 | 全部可用 | 全部可用 |

**Q: Creative Pack 值得购买吗？**

A: 取决于你的目标：
- **只想玩别人的世界** - 不需要，免费版足够
- **想创建简单世界** - 不需要，免费版可以创建 2 个世界
- **想分享创作或制作复杂冒险** - 值得购买

**Q: 有哪些 DLC 可用？**

A: 主要 DLC：
- **Creative Pack** - 创作功能扩展（如上表）
- **Epic Mounts** - 更多坐骑
- 未来可能推出更多内容包

**Q: 游戏还在更新吗？**

A: 活跃状态。MyWorld 标记为"无尽标题"（Endless Title），表示持续开发中。但更新频率和内容规模不如 2017-2018 年活跃期。

## 版本限制与 DLC

### 🆓 免费版限制

MyWorld 采用免费游玩 + DLC 扩展的商业模式。免费版有以下限制：

| 限制 | 描述 | 影响 |
|------|------|------|
| **世界数量** | 最多 2 个 | 无法创建大型冒险 |
| **世界链接** | 不支持 | 无法创建章节式游戏 |
| **Workshop 上传** | 不支持 | 无法分享社区 |
| **敌人创建器** | 不支持 | 只能使用预设敌人 |
| **对象库** | 350+ 物品 | 创造多样性有限 |
| **多人游戏** | 完全支持 | 可加入他人游戏 |

**免费版适合**：
- 玩家 - 下载并游玩 Workshop 世界
- 测试者 - 尝试创作，决定是否购买 DLC

**免费版不适合**：
- 严肃创作者 - 受限太多
- 社区贡献者 - 无法分享作品

### 💎 Creative Pack DLC

**价格**：约 $4.99 - $9.99（根据促销）

**解锁功能**：

✅ **无限制世界创建**
- 不再有 2 个世界的限制
- 创建尽可能多的世界
- 构建大型多世界冒险

✅ **世界链接系统**
- 通过传送门链接多个世界
- 创建章节式 RPG 冒险
- 打包上传到 Workshop

✅ **敌人创建器**
- 模块化混合身体部件
- 自定义敌人属性和外观
- 创建独特 Boss 和怪物

✅ **扩展对象库**
- 从 350+ 扩展到 900+ 物品
- 更多建筑、装饰、NPC、敌人
- 9 种世界主题全部可用

✅ **Workshop 上传**
- 分享创作到 Steam 社区
- 获取玩家反馈和下载
- 参与社区生态

**购买建议**：
- ✅ **强烈推荐** - 如果你想认真创作
- ✅ **值得** - 如果你计划分享作品
- ⚠️ **可选** - 如果你只打算游玩
- ❌ **不需要** - 如果只是测试游戏

## 结语

MyWorld 是一款雄心勃勃的动作 RPG 创作工具。它通过**降低创作门槛**（无代码、直观编辑器）和**创新性功能**（传送门链接、Hubworld 空间菜单、模块化敌人创建），在 RPG 创作者领域占据独特生态位。

虽然游戏在某些方面仍有不足（平台限制、深度不够、战斗手感），但其核心理念——**模糊创造与游玩的界限**——依然具有前瞻性。对于想要创建 3D 动作 RPG 但缺乏编程技能的爱好者来说，MyWorld 提供了一个宝贵的起点。

- 🛠️ 作为创作者，掌握世界创建工具，设计出色的冒险
- 🤝 作为社区成员，理解游戏生态，贡献优质内容

祝您在 MyWorld 中创造精彩的世界和难忘的冒险！
